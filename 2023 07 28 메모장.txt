추상 : 공통적 추출 / 개별적 제외
추상 클래스 : 클래스를 미완성으로 만들어 놓음

자바는 다중상속이 불가하지만 인터페이스를 이용하면 가능하다

인터페이스는 설계만 해야함. 구현X

인터페이스 == 접점

자바에서의 인터페이스 == 클래스간의 접점 -> 클래스가 공통적으로 가져야하는 필드, 메소드를 모아두고 상속

시스템 에러 == 버그

try{
	예외(에러)가 발생할 것 같은 코드
}catch(Exception e) {
	System.out.print(e + "해당 에러 발생")
} finally {
	에러가 나든 안나든 무조건 수행
}
<-- finally는 안써도 됨
