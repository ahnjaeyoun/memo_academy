git clone : git에 저장되어있는 것을 복사해오기
git pull : git에 저장되어있는 것을 가져오기(컴퓨터에 있는 코딩파일에 덮어씌우기)
ls : 폴더안에 있는 작업 파일을 보여줌
cd src : src 폴더로 이동(src 말고도 다른 폴더가 있다면 이동할 수 있다)
cd .. : 상위 폴더로 이동

run class
(run할 파일이름) (변수 이름 지정) = new (run할 파일이름)();
