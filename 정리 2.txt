DDL
ALTER TABLE ~ ADD(~); 컬럼 추가
ALTER TABLE ~ MODIFY ~ ; 컬럼 수정
ALTER TABLE ~ DROP~; 열삭제
CREATE TABLE ~ AS SELECT * FROM ~;
DROP TABLE ~ CASCADE CONSTRAINTS; DROP TABLE 삭제옵션 CASCADE CONSTRAINTS 사용
		--> 삭제하려는 테이블과 연결된 FK 제약조건을 모두 삭제 
TRUNCATE TABLE ~; 테이블 행, 열 완전히 삭제
DDL과 DML 구문을 섞어서 수행하면 안된다.


DML
INSERT INTO ~ VALUES~; 컬럼 추가

[WHERE 컬럼명 비교연산자 비교값]; 컬럼 값 수정
